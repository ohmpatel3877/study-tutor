# StudyTutor — Adaptive Exam Prep

Two powerful study tools with built-in tutoring features:

## 📚 [StudyTutor (Generalized)](https://ohmpatel3877.github.io/study-tutor/)
The general-purpose adaptive exam prep tool. Configure it for any course:
- Import/export questions as JSON
- Spaced repetition (SM-2 algorithm)
- Socratic hint system
- Confidence tracking
- Topic mastery insights
- Multiple study modes

## 🔬 [Histo/Cytology Exam Prep](https://ohmpatel3877.github.io/study-tutor/histo-cytology-exam-prep.html)
Specialized for Michener 2nd Year Histo/Cytology with 150+ questions across 16 topics.

## Features
- 🧠 **Spaced Repetition** — SM-2 algorithm schedules reviews
- 💡 **Socratic Hints** — Progressive hints guide you to answers
- 📊 **Learning Insights** — Track mastery per topic, confidence calibration
- 🎯 **Adaptive Review** — Prioritizes weak topics and overdue questions
- 🃏 **Flashcards** — Flip-to-reveal with spaced review
- 🎨 **Multiple Themes** — Purple, Coffee, Forest, Ocean, Sunset, Lavender, Slate
- ⌨️ **Keyboard Shortcuts** — 1-4 to answer, arrows to navigate
- 💾 **Persistent State** — Progress saved in browser localStorage

## Quick Start for Custom Courses
1. Open StudyTutor
2. Go to Setup tab
3. Replace questions or import a JSON file
4. Configure topics and icons
5. Save and study!
